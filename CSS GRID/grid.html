>>(해결)그냥 gird template로 42칸을 나눠놓았으면 각각의 heart div에 grid-row와 colum으로 범위지정하면되는데 과제할당시에는 생각이 안났던거같음
  • css-flex
  • flex는 한방향 레이아웃 시스템(1차원)
  • flex-direction을 통해 메인축의 방향을 결정할수있음(기본row)
  • flex container에 justify-content등을 통해 item들을 정렬할 수 있음
  • flex item에는 flex-basis,grow등을 통해 아이템의 크기 설정 가능
  • css-grid
  • 두 방향 레이아웃 시스템 (2차원)
  • flex와 같이 컨테이너와 아이템에 속성적용
  • grid container는 grid-template-rows,colums를 통해 형태정의
  • 정렬은 flex와 유사하게 가능
  • grid item은 각 셀의 영역을 지정하여 grid라인 번호를통해 row와 colum의 범위 결정가능
